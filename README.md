# Taller Estrategia de Ecosistemas Móviles 2026

Este proyecto analiza:

1. Monetización Android vs iOS
2. Fragmentación del sistema operativo
3. Perfiles de usuario
4. Seguridad empresarial
5. Comparación de asistentes IA

El código fue desarrollado en Dart utilizando:
- List<Map<String, dynamic>>
- Ciclos forEach
- Condicionales if
- Filtros where
- Método contains
